## 属性一览

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `src` | `string` | — | Excalidraw JSON 文件路径 |
| `theme` | `'light' \| 'dark'` | `'light'` | 主题模式 |
| `height` | `number \| string` | 自动 | 图表高度 |
| `editable` | `boolean` | `DEV` 模式 | 是否可编辑 |
| `gridModeEnabled` | `boolean` | `false` | 网格模式 |
| `zenModeEnabled` | `boolean` | `false` | 禅模式 |
| `saveEndpoint` | `string` | `/__excalidraw_save` | 保存接口地址 |
